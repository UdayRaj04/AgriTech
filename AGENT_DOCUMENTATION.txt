PROJECT OVERVIEW:
================================================================================
AgriTech AI is a comprehensive agricultural AI system that empowers farmers with 
intelligent decision-making tools for crop management, fertilizer optimization, 
and disease detection. The system consists of three main AI-powered features.

DEVELOPER: Uday Raj
TECHNOLOGY: React (Frontend) + FastAPI (Backend) + TensorFlow/Scikit-learn (ML)

1. üå± CROP RECOMMENDATION SYSTEM
================================================================================
Purpose: Suggests optimal crops based on soil and climate analysis
Model: Random Forest Classifier
Accuracy: 99%

INPUT PARAMETERS:
- Nitrogen (N): 0-150 kg/hectare
- Phosphorus (P): 0-150 kg/hectare  
- Potassium (K): 0-150 kg/hectare
- Temperature: 0-50¬∞C
- Humidity: 0-100%
- Soil pH: 0-14
- Rainfall: 0-300mm

SUPPORTED CROPS (22 varieties):
Rice, Maize, Jute, Cotton, Coconut, Papaya, Orange, Apple, Muskmelon, 
Watermelon, Grapes, Mango, Banana, Pomegranate, Lentil, Blackgram, 
Mungbean, Mothbeans, Pigeonpeas, Kidneybeans, Chickpea, Coffee

OUTPUT: Top 3 crop recommendations with confidence scores

API ENDPOINT: POST https://argi.in/crop-recom
Request Body:
{
  "nitrogen": float,
  "phosphorus": float, 
  "potassium": float,
  "temperature": float,
  "humidity": float,
  "ph": float,
  "rainfall": float
}

Response:
{
  "prediction": "Rice",
  "confidence": 0.95,
  "alternatives": ["Rice", "Wheat", "Barley"]
}

                              API INTEGRATION EXAMPLES

EXAMPLE 1: Crop Recommendation Request
=====================================
User: "I have soil with N=80, P=45, K=60, temp=25¬∞C, humidity=60%, pH=6.5, rainfall=100mm. What crops should I plant?"

Agent Action:
1. Collect all required parameters
2. POST to https://argi.in/crop-recom
3. Present top 3 recommendations with confidence
4. Provide additional crop information if requested

EXAMPLE 2: Disease Detection Request
===================================
User: "I have tomato leaves with dark spots. Can you identify the disease?"

Agent Action:
1. Ask user to upload leaf image
2. POST image to https://argi.in/disease-detect
3. Provide diagnosis with detailed treatment information
4. Suggest preventive measures

EXAMPLE 3: Fertilizer Recommendation
===================================
User: "I'm growing wheat in loamy soil. What fertilizer should I use?"

Agent Action:
1. Collect all fertilizer parameters
2. POST to https://argi.in/ferti-recom
3. Provide fertilizer recommendation
4. Explain application timing and methods
